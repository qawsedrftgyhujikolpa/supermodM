FabricOpt - PoC optimization mod
================================

This is a minimal Fabric mod scaffold and a PoC that registers a server tick listener.
Target: Fabric, Minecraft 1.20.4

Files created:
- src/main/java/com/example/fabricopt/FabricOptMod.java
- src/main/resources/fabric.mod.json
- config/fabricopt.properties

Build with Gradle (requires Fabric Loom and proper versions).
