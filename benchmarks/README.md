ベンチマーク概要

目的:
このMODがパフォーマンスに与える影響を定量化するための手順とツール群。

主要シナリオ:
- 空ワールド（プレイヤー無し・チャンク生成負荷なし）
- エンティティストレス（大量のモブ/アイテム）
- チャンクロード（プレイヤー移動によるチャンク生成/破棄）
- レッドストーン/ティック負荷（クロックや機構）

測定指標:
- TPS（サーバ）
- FPS（クライアント）
- CPU使用率, メモリ使用量
- GC頻度と停止時間（可能ならば）
- 平均ティック時間（ms/t）

ワークフロー（簡易）:
1. 同一マシン・同設定でベースライン（MOD無し）を計測
2. Mod有りで同じシナリオを複数回実行して平均を取る
3. CSVで結果を保存し、差分を評価する

ツール:
- measure_tps.ps1: サーバログからTPSやms/tを解析する簡易スクリプト
- Windows の場合: Get-Process -Name java でCPU/メモリスナップショット

注意:
- 環境差（JVMオプション、CPU、OS）は結果に大きく影響するため、比較は必ず同一環境で行うこと。
