参考MODのライセンス確認手順

目的:
- 既存MODを参考にする際の法的リスクを最小化する
- コードの直接コピーを避け、必要な帰属や許可を確認する

推奨チェックリスト:
1. ソースリポジトリ（GitHub等）で LICENSE ファイルを確認する。
2. mod のリリースアーティファクト（.jar）を解凍して META-INF や LICENSE を確認する。
3. ライセンス種別（MIT / Apache-2.0 / GPL / Proprietary 等）を記録する。
4. 再配布・改変・特許条項の可否を確認する（Apache は特許条項に注意）。
5. Proprietary（プロプライエタリ）やライセンス不明のものはコードを直接写さない。
6. 参考にしたアルゴリズムやアイデアは README 等に明記し、必要な帰属を追加する。

候補MOD（作業メモ、要手動確認）:
- Sodium — リポジトリ/リリースでLICENSEを確認してください。
- Phosphor — 同上。
- Lithium — 同上。
- OptiFine — 多くの配布物はプロプライエタリ／独自条件のため注意が必要です。

実務上の勧め:
- 他MODのソースをそのままコピーしないこと。アルゴリズムから学び、自分で実装する。
- 必要なら法務または著作権に詳しい人に相談する。
- 参考元はドキュメントで明示する（透明性）。
